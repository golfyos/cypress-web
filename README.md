# Cypress Automate Web
Automate Web Script using Typescript language.

### Installation
```sh
npm install
```

### Open development environment
```sh
npm run cypress:open

// or shorthand
npm run o
```

### Run All Test suites
```sh
npm run cypress:run

// or shorthand
npm run r
```

### Generate Report from Previous Run
```sh
npm run report
```

## Run, Generate Report, Open Report
```sh
npm run run-report
npm run report:show
```
